## Features

- Shorten long URLs
- Redirect short URLs to the original URLs
- View and manage shortened URLs via Django admin
- Run and view test results from the web interface

## Technologies Used

- Python 3.9
- Django 3.2
- PostgreSQL
- Docker
- Docker Compose
- Bootstrap 4
- pytest
